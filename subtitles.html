Samuel Ting Subtitles

About this project

This project is a continuation of our previous project here. Our goal is to transcribe the entirety of Nobel Prize physicist Dr Samuel Ting’s Science Cafe hosted by Cary Memorial. We aim to accomplish this by using a combination of manual transcription and machine learning based captions services.


Purpose of this project

The purpose of this project is to make Dr. Ting's talk more accessible to the wider audience, particularly deaf or non-fluent speakers of English.


Project Timeline (may be subject to change)
  • March - April 2022

    We will finish writing the script which automatically creates subtitles for the lecture using google’s speech to text api and machine learning. Finish the subtitles by hand and begin analysis between machine learning subtitles.

  • April - May 2022

    Wrap up and compile the subtitles into a text file and upload it to the lecture itself along with an analysis of the API’s subtitles, so that the video would be more accessible to the public who are not fluent in English or need assistance while watching videos.

  • May - Mid-May 2022

    Translate the subtitles into more languages such as Mandarin, Spanish, French, Japanese, German, etc. And upload those accurately translated files using machine learning to the video so that it will become more inclusive to those who do not have English as their first language but are still interested in science.
